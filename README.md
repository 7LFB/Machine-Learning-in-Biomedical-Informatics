## 1. 课程简介 （建设中...）

### 1.1 时代背景与战略意义

近日，国务院正式印发《关于深入实施"人工智能+"行动的意见》，标志着我国人工智能发展实现从技术突破向全要素赋能的关键跃升。作为继"互联网+"之后国家推动技术革命和产业融合的又一战略部署，"人工智能+"行动立足我国产业体系完备、市场规模庞大、应用场景丰富三大核心优势，构建"创新带应用、应用促创新"的螺旋式发展范式，加速人工智能与实体经济全链重构、深向耦合，全面激发产业变革新动能、培育智能经济新范式。

在"人工智能+"战略框架下，**生物医学领域是最具变革潜力和社会价值的重点应用方向之一**。从生产力维度来看，人工智能技术可以通过算法优化与数据要素价值释放，重构生物医学研究与医疗服务的全要素生产率提升路径，有效降低新药研发、疾病诊断、精准治疗等传统生产要素的边际成本约束，为健康中国战略和生命科学高质量发展注入新动能。从应用场景来看，生物医学领域拥有海量的异构数据（基因组、蛋白质组、医学影像、电子病历等）、复杂的科学问题（蛋白质结构预测、药物设计、疾病机制解析）以及迫切的社会需求（精准医疗、新药研发、疫情防控），为人工智能技术提供了最丰富的应用土壤和最严格的实战检验。

### 1.2 课程定位与培养目标

生物医学信息学是一门融合计算机科学、数据科学、人工智能与生物医学的新兴交叉学科，致力于运用先进的计算技术和算法解决生物医学领域的复杂问题。本课程响应国家**"人工智能+"行动**、**"新医科"建设**及**"医工交叉融合"**的政策导向，依托于学校在计算机科学、人工智能和大数据技术方面的学科优势，面向计算机相关专业学生开设，旨在培养既懂计算技术又了解生物医学应用场景的复合型人才。

随着高通量测序技术、医学影像技术、可穿戴设备和电子健康档案系统的快速发展，生物医学领域正在产生海量的异构数据，对计算机专业人才提出了迫切需求。从AlphaFold2突破蛋白质结构预测50年难题，到深度学习在医学影像诊断中超越人类专家水平，从大语言模型辅助临床决策，到AI驱动的新药研发周期缩短数年——人工智能正在深刻改变生物医学研究与医疗实践的范式。**这一领域不仅是"人工智能+"战略的重要落地场景，更是计算机专业学生施展才华、创造社会价值的广阔舞台。**

本课程面向计算机科学与技术、软件工程、数据科学、人工智能等专业的本科生，一般在本科大三或大四阶段进行讲授，目前共设48个课时，分16周讲授。课程从计算机专业学生的知识背景出发，**前半部分系统讲解生物医学信息学的基础知识、经典算法和数据结构**，**后半部分按照基因组学、蛋白质组学、医学影像、临床信息学等不同领域，深入讲解深度学习等前沿计算技术的应用**，并精选顶级会议和期刊的前沿文献供学生进行拓展学习。

### 1.3 课程特色与教学理念

课程目的是帮助计算机专业学生理解生物医学问题的计算本质，掌握从经典算法到深度学习的完整技术栈，培养跨学科问题求解能力和"人工智能+"时代的创新思维。课程要求学生具备扎实的数据结构与算法基础，掌握至少一门编程语言（Python、Java、C++等），了解基本的机器学习和深度学习知识。

本课程采用的PPT教案为中英文双语，讲授为双语教学。教学内容参考了CMU、斯坦福大学、MIT等顶尖计算机院系开设的计算生物学和医疗健康信息学课程，同时结合国内生物医学研究和医疗信息化的实际需求进行了优化设计。课程强调**"问题驱动、算法为核心、工程实践为导向"**的教学理念，每个主题都配有真实数据集、开源工具和编程项目，帮助学生从计算机专业视角理解和解决生物医学问题。

**课程核心理念：**
- **战略引领**：紧扣"人工智能+"国家战略，培养面向未来的复合型人才
- **系统培养**：从经典算法到深度学习的完整知识体系
- **领域深耕**：按生物医学不同领域组织深度学习应用，实现"人工智能+生物医学"的深度融合
- **工程导向**：强调代码实现、工具使用和系统开发，打通"实验室—市场"转化通道
- **前沿追踪**：紧跟AlphaFold、ChatGPT等最新技术进展，培养创新能力
- **实战驱动**：真实数据集、开源项目、竞赛驱动学习，破解技术落地"最后一公里"难题

### 1.4 职业发展与产业机会

在"人工智能+"战略推动下，生物医学信息学领域正迎来前所未有的发展机遇：

**产业需求旺盛**：AI制药公司（如英矽智能、晶泰科技）、医疗影像AI企业（如推想科技、数坤科技）、基因检测公司（如华大基因、贝瑞基因）、互联网医疗平台（如平安好医生、微医）等新兴企业快速崛起，对计算机专业人才需求激增。

**职业前景广阔**：生物信息学工程师、医疗AI算法工程师、计算生物学家、精准医疗数据科学家等新兴职业成为高薪热门岗位，薪资水平普遍高于传统IT行业。

**社会价值显著**：通过技术创新助力疾病诊断、新药研发、精准治疗，直接造福人类健康，实现个人价值与社会价值的统一。

**创新创业机遇**：国家大力支持生物医学领域的科技创新和成果转化，为有志于创业的学生提供了丰富的政策支持和市场空间。

## 9. 课程考核方式

- **平时作业** (25%): 8次编程作业 (课时2-8基础部分)
- **课程项目** (45%): 4次专题项目 (课时9-15应用部分) + 1次期末综合项目
- **论文阅读报告** (15%): 每个应用领域精读1篇顶会/顶刊论文，撰写技术报告
- **课堂参与** (15%): 出勤、讨论、代码审查、同行评议

### 期末项目要求
学生可选择以下方向之一（鼓励结合"人工智能+"战略，选择具有实际应用价值的题目）：

1. **论文复现**: 复现某篇顶会论文并在新数据集上验证
2. **方法改进**: 对现有方法提出改进并进行实验验证
3. **应用开发**: 开发完整的生物医学信息学应用系统
4. **竞赛参与**: 参加Kaggle等平台的相关竞赛并取得成绩
5. **创新研究**: 提出原创性研究思路并进行初步验证
